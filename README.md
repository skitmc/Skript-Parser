# Skript-Parser

An advanced, web-based code editor and syntax highlighter for the **Skript** scripting language.

## 🚀 Features
- **Monaco Engine:** Uses the same engine as VS Code for a premium editing experience.
- **Syntax Highlighting:** Custom-built definitions for Skript triggers, effects, and variables.
- **Project Scope:** Built to assist and integrate with the ongoing work at **SKITMC**.

## 🛠️ How to Use
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Start writing scripts!
